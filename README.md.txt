# Tic Tac Toe Game

This project is a **Tic Tac Toe Web Application** developed as **Task-03** for the  
**Prodigy InfoTech Web Development Internship**.

## Project Description
The Tic Tac Toe game allows two players to play alternately as **X** and **O** on a
3Ã—3 grid. The application automatically detects **win and draw conditions** and
provides an option to **restart the game**.

## Features
- Two-player gameplay (X and O)
- Automatic win detection
- Draw detection
- Restart game option
- Clean and user-friendly interface
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
